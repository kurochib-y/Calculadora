A calculadora desenvolvida em Portugol tem como objetivo realizar operações matemáticas básicas de forma rápida e prática. O programa permite que o usuário escolha qual operação deseja executar, como adição, subtração, multiplicação ou divisão.

Inicialmente, o programa apresenta um menu com as opções disponíveis. Em seguida, o usuário escolhe a operação desejada e informa os dois números que serão utilizados no cálculo. O sistema realiza a operação selecionada e exibe o resultado na tela.
